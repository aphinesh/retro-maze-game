Retro Maze Game (Raspberry Pi & Pygame)

📌 Overview
A retro-style maze game built in Python using Pygame, where players navigate through a puzzle-based environment by collecting items, avoiding hazards, and unlocking progression paths. The game includes both a traditional keyboard-controlled version and a Raspberry Pi version featuring physical inputs and LED feedback for a more immersive experience.

✨ Features
- Inventory system with item selection and usage
- Interactive objects (doors, puzzles, notes)
- Hazard mechanics (lasers, obstacles)
- Timer-based gameplay
- Lives/health system
- Two interaction modes: keyboard and tangible controls

🛠 Technologies Used
- Python
- Pygame
- Raspberry Pi (GPIO)

▶️ How to Run
STANDARD VERSION (PC):
- Install Python (3.x)
Install Pygame:
- pip install pygame
Run the game:
- python main.py
RASPBERRY PI VERSION:
- Requires a Raspberry Pi with GPIO setup
- Physical inputs (buttons/switches) mapped to game controls
- LED outputs provide visual feedback during gameplay

🎮 Controls (Keyboard Version)
- Movement: WASD or Arrow Keys
- Interact / Select: ENTER
- Pick up item: P
- Inventory: I
- Exit: DELETE or ESC

🎯 Project Purpose
This project was developed to investigate how different interaction methods affect player experience. It compares traditional keyboard input with tangible interaction using a Raspberry Pi to evaluate differences in usability, performance, and engagement.
